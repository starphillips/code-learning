
**Pseudo**

I want to search through the array so that it takes the first number and checks if this can be added to create the value of the target, if not it will check the next number in the array and the numbers following this.

I would use a **for loop**

For the first number in the array,

add this to the next index value in the array until it reaches the target.

If they do not reach the target, pick the next indexâ€™s value of array, and add this to next index value until it reaches target.

Return the indexes of the numbers in the array that reach the target

```ts
function twoSum(nums: number[], target: number): number[] {

    for (let i = 0; i < nums.length; i++) {
        for(let j = i+1; j < nums.length; j++){
            if (nums[i] + nums[j] == target) {
                return [i,j];
            }
        }
    }
};
```
