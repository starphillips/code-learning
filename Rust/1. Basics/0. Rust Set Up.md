Download Rust online

`rustc --version`

If this doesnt work, update your environment variables with an additional path

This path will be where Rust was installed on your machine e.g. 

`C:\Users\starkie.phillips\.cargo\bin`

Download Rust extension in your Code Editor (VSC)

##### Running a program

`rustc your.file.name.rs`

This will create an exe for your file

`./main` 


##### Check cargo 
Package Manager / Build System = cargo

`cargo --version`

run `cargo new your_cargo_name` to create this package.

cargo is like the same as packge.json
