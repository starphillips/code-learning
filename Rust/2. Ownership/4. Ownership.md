
#### A way to manage memory

Ownership is a set of rules that govern how a Rust program manages memory.

Cares about run time performance because its a system program. 
It programs low-level things where runtime is crucial and caring about memory is crucial. 


1. Each value in Rust has a variable that's called its owner
2. There can only be one owner at a time of said variable
3. When the owner goes out of scope, the value will be dropped
